A portfolio and playground built with Angular — a space to showcase digital illustrations and 3D art while learning the framework.

## Getting started

```bash
npm install
npm start
```

Open `http://localhost:4200`.

## Scripts

| Command | Description |
|---|---|
| `npm start` | Start the dev server |
| `npm test` | Run tests in watch mode |
| `npm run test:ci` | Run tests once (no watch) |
| `npm run lint` | Lint the codebase |
| `npm run build` | Production build |

## CI

Every pull request to `main` runs lint, tests, and build. See [`docs/pipeline.md`](docs/pipeline.md) for details.
